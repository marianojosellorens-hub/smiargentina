## Stack tecnológico

| Capa | Tecnología |
|------|------------|
| Framework | TanStack Start |
| Frontend | React 19, TanStack Router v1 |
| Build | Vite 7 |
| Estilos | Tailwind CSS 4 + CSS custom properties |
| Tipografía | Google Fonts — Oswald + Lato |
| Lenguaje | TypeScript 5.7 (strict mode) |
| Despliegue | Netlify |

## Cómo ejecutar localmente

```bash
# Instalar dependencias
npm install

# Servidor de desarrollo
npm run dev

# Build de producción
npm run build

# Preview del build
npm run preview
```

El servidor de desarrollo estará disponible en `http://localhost:3000` (o vía Netlify CLI en `http://localhost:8888`).

## Variables de entorno

No se requieren variables de entorno para la funcionalidad base del sitio.
